Title: Mifos X Web App - Create the create tax component and design its user interface
Description: ## Objective
The objective is to create the create tax component and design its user interface.

### Description
1. Create the create tax component under the products -> manage tax configurations -> manage tax components section.
2. Connect it via routing from create tax component button in manage tax components component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the create tax component using angular material components and flex layout directives to ensure responsiveness.
5. Implement the required API so a new tax can be created. Make sure after tax creation we are redirected back to tax list view if view tax component does not exist at the time of implementation of this component.
6. Write comments for the implemented component.

#### References
You may have a look at the current implemented components in `accounting`, `system` or `products` section for reference.
