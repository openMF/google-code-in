Title: Mifos X Web App - Create the view employee component and design its user interface
Description: ## Objective
The objective is to create the view employee component and design its user interface.

### Description
1. Create the view empoyee component under the organization -> manage employees section.
2. Connect it via routing from employees table in employees component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the view user component using angular material components and flex layout directives to ensure responsiveness. (Use Angular Material Typography instead of SCSS wherever possible, refer more recently created components for reference)
5. Fetch the data of the employee by implementing the required API.
6. Write comments for the implemented component.

#### References
You may have a look at the current implemented components in `accounting`, `system` or `products` section for reference.
