Title: Mifos - Create a fixed deposit products component and design its user interface in Mifos X Web App
Description: ## Objective
The objective is to create a fixed deposit products component and design its user interface.

### Description
1. Create a fixed deposit products component under the products section.
2. Connect it via routing from the products component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the fixed deposit products component using angular material components and flex layout directives to ensure responsiveness.
