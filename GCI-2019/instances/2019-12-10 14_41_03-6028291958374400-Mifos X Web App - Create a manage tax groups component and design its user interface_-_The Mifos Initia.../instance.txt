Title: Mifos X Web App - Create a manage tax groups component and design its user interface
Description: ## Objective
The objective is to create a manage tax groups component and design its user interface.

### Description
1. Create a manage tax groups component under the products section.
2. Connect it via routing from products -> manage tax configurations component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the manage tax configurations component using angular material components and flex layout directives to ensure responsiveness.
5. Fetch the data by implementing the required API and populate the table. Make sure searching and sorting works.
6. Write comments for the implemented component.

#### References
You may have a look at the current implemented component `products -> manage tax components` for reference.
