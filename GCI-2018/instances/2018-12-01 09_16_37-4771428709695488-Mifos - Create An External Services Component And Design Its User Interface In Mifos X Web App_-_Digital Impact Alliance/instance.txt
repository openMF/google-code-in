Title: Mifos - Create An External Services Component And Design Its User Interface In Mifos X Web App
Description: ## Objective
The objective is to create an external services component and design its user interface.

### Description
1. Create an external services component under the system section.
2. Connect it via routing from the system component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the external services component using angular material components and flex layout directives to ensure responsiveness.
