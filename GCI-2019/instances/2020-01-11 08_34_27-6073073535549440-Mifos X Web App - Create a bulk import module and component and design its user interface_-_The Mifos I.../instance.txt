Title: Mifos X Web App - Create a bulk import module and component and design its user interface
Description: ## Objective
The objective is to create a bulk import module and component and design its user interface.

### Description
1. Create a bulk import module and component in the app folder.
2. Connect the component via routing from organization -> bulk import so that its accessible. **Note:** Explore the possibility of lazy loading this module and implement accordingly.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the bulk import component using angular material components and flex layout directives to ensure responsiveness.
5. Write comments for the implemented module and component.

#### References
You may have a look at the current implemented components in `accounting`, `system` or `products` section for reference.
