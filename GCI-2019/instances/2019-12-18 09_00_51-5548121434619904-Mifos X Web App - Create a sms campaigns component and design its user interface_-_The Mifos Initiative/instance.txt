Title: Mifos X Web App - Create a sms campaigns component and design its user interface
Description: ## Objective
The objective is to create a sms campaigns component and design its user interface.

### Description
1. Create a sms campaigns component under the organization section.
2. Connect it via routing from organization component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the sms campaigns component using angular material components and flex layout directives to ensure responsiveness.
5. Fetch the data by implementing the required API and populate the table. Make sure searching and sorting works.
6. Write comments for the implemented component.

#### References
You may have a look at the current implemented components in `accounting`, `system` or `products` section for reference.
