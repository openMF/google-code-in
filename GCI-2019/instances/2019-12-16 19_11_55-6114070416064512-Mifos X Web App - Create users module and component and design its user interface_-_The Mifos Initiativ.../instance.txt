Title: Mifos X Web App - Create users module and component and design its user interface
Description: ## Objective
The objective is to create users module and component and design its user interface.

### Description
1. Create a users module and component in the app folder.
2. Connect the component via routing from the toolbar component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the users component using angular material components and flex layout directives to ensure responsiveness.
5. Fetch the data by implementing the required API and populate the table. Make sure searching and sorting works.
6. Write comments for the implemented module and component.

#### References
You may have a look at the current implemented components in `accounting`, `system` or `products` section for reference.

## How to get help 
If you need help, be sure to post your question to our [developer mailing list](http://mifos.org/resources/community/communications/#mailing-lists) or ask in our [chatroom](https://gitter.im/openMF/web-app).

All the best! Do remember, quality over quantity is the key.
