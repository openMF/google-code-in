Title: Mifos X Web App - Create a tellers component and design its user interface
Description: ## Objective
The objective is to create a tellers component and design its user interface.

### Description
1. Create a tellers component under the organization section.
2. Connect it via routing from organization component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the tellers component using angular material components and flex layout directives to ensure responsiveness.
5. Fetch the data by implementing the required API and populate the table. Make sure searching and sorting works.
6. Remove the edit/delete buttons and rename view button to `view cashiers`. 
7. Write comments for the implemented component.

#### References
You may have a look at the current implemented components in `accounting`, `system` or `products` section for reference.
