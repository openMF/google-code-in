Title: Mifos X Web App - Create the view charge component and design its user interface
Description: ## Objective
The objective is to create the view charge component and design its user interface.

### Description
1. Create the view charge component under the products -> charges section.
2. Connect it via routing from charges table in charges component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the view charge component using angular material components and flex layout directives to ensure responsiveness. (Use Angular Material Typography instead of SCSS wherever possible, refer more recently created components for reference)
5. Fetch the data of charge product by implementing the required API.
6. Implement required API for deletion of charge product. 
7. Write comments for the implemented component.

#### References
You may have a look at the current implemented components in `accounting`, `system` or `products` section for reference.
