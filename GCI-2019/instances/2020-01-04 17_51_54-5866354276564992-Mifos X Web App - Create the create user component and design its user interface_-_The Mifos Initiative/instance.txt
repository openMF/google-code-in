Title: Mifos X Web App - Create the create user component and design its user interface
Description: ## Objective
The objective is to create the create user component and design its user interface.

### Description
1. Create the create user component under the users section.
2. Connect it via routing from create user button in users component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the create user component using angular material components and flex layout directives to ensure responsiveness. (Please read issue description for more information)
5. Implement the required API so a new user can be created. Make sure after user creation we are redirected back to users list view if view user component does not exist at the time of implementation of this component.
6. Write comments for the implemented component.

#### References
You may have a look at the current implemented components in `accounting`, `system` or `products` section for reference.
