Title: Mifos X Web App - Create a working days component and design its user interface
Description: ## Objective
The objective is to create a working days component and design its user interface.

### Description
1. Create a working days component under the organization section.
2. Connect it via routing from organization component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the working days component using angular material components and flex layout directives to ensure responsiveness.
5. Allow configuration of working days and other available options by implementing the required API.
6. Write comments for the implemented component.

#### References
You may have a look at the current implemented components in `accounting`, `system` or `products` section for reference.
