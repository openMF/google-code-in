Title: Mifos X Web App - Create a view template component and design its user interface
Description: ## Objective
The objective is to create a view template component and design its user interface.

### Description
1. Create a view template under the templates section.
2. Connect it via routing from templates component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the view template component using angular material components and flex layout directives to ensure responsiveness.
5. Implement the required API to show the data associated with the template. (See System -> Account Number Preferences -> View as an example)
6. Implement the required API to allow deletion of a template.
7. Write comments for the implemented component.

#### References
You may have a look at the current implemented components in `accounting`, `system` or `products` section for reference.

## How to get help 
If you need help, be sure to post your question to our [developer mailing list](http://mifos.org/resources/community/communications/#mailing-lists) or ask in our [chatroom](https://gitter.im/openMF/web-app).

All the best! Do remember, quality over quantity is the key.
