Title: Mifos - Create A Surveys Component And Design Its User Interface In Mifos X Web App
Description: ## Objective
The objective is to create a surveys component and design its user interface.

### Description
1. Create a surveys component under the system section.
2. Connect it via routing from the system component so that its accessible.
3. Make sure to read the guidelines to replicate a screen from [here](https://github.com/openMF/web-app/wiki/Replicate-an-existing-Community-App-screen-into-the-new-Web-App).
4. Design the user interface for the surveys component using angular material components and flex layout directives to ensure responsiveness.

You may have a look at the current implemented components in the `accounting` section for reference.
